使用包管理器 ​

我们建议您使用包管理器（如 NPM、Yarn 或 pnpm）安装 Dinert Element Plus.

用法 ​

shell
# 选择一个你喜欢的包管理器

# NPM
$ npm install lodash ant-design @dinert/ant-design

# Yarn
$ yarn add lodash ant-design @dinert/ant-design

# pnpm
$ pnpm install lodash ant-design @dinert/ant-design
